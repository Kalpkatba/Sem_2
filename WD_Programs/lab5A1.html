Concept of WWW

  • It stands for World Wide Web
  • It is basically a system of internet servers that support specially formatted documents
  • It is a collection of text pages,videos,digital photohraphs and animations we can access over the internet.
  • User access the World Wide Web facilities via client call a web brouser.

HTTP Protocol

  1. HTTP stands for Hyper Text Transfer Protocol.
  2. HTTP is based on the client-server architechture model and a stateless request/response protocol
  3. 404 File Not Found is a common HTTP status code.
